# Getting started

This is a quick start guide to start using ULTRA to install packages.

```bash
npm install -g ultrapkg
```

## Usage

To install packages from `package.json` you have to run the following command:

```bash
ultra install
```

You can also install packages from the command line:

```bash
ultra install lodash # You can also use flags like -D
```
